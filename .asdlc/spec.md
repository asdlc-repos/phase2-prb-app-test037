# Overview

A minimal API service that returns the current server time in JSON format when called.

# Personas

- Developer — calls the endpoint to get the current time.

# Features

- A developer sends a GET request to the endpoint.
- The API responds with the current time in JSON format.
- The service runs and accepts HTTP requests.
- The response includes a timestamp field with the current date and time.
- The API returns a successful status code when the request is valid.
